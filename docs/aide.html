Tutoriels rapides

1

Comment lire mes statistiques ?

Sur votre dashboard enquêteur, vous trouverez plusieurs indicateurs :

  • Questionnaires soumis : Nombre total de formulaires envoyés
  • Taux de validation : Pourcentage de questionnaires acceptés
  • Questionnaires en attente : En cours de vérification par le superviseur
  • Questionnaires rejetés : À corriger et renvoyer
Consultez le graphique d'évolution pour voir votre progression dans le temps.
2

Que faire si mon questionnaire est rejeté ?

Lorsqu'un questionnaire est rejeté par le superviseur :

  1. Consultez la section "Questionnaires rejetés" de votre dashboard
  2. Cliquez sur le questionnaire pour voir les commentaires du superviseur
  3. Notez les erreurs signalées (champs manquants, incohérences, etc.)
  4. Ouvrez ODK Collect et modifiez le formulaire concerné
  5. Corrigez les erreurs identifiées
  6. Soumettez à nouveau le questionnaire
Attention : Un questionnaire rejeté plus de 3 fois sera définitivement supprimé.
3

Comment télécharger le questionnaire ?

Pour télécharger le formulaire ODK sur votre tablette/téléphone :

  1. Ouvrez l'application ODK Collect
  2. Allez dans Paramètres → Serveur
  3. Configurez l'URL du serveur (fournie par votre superviseur)
  4. Retournez au menu principal
  5. Cliquez sur "Obtenir un formulaire vide"
  6. Sélectionnez le questionnaire et cliquez sur "Télécharger"
Le questionnaire XLSForm est également disponible dans la section "Notes Techniques".
4

Comment synchroniser mes données ?

La synchronisation permet d'envoyer vos questionnaires complétés au serveur :

  1. Assurez-vous d'avoir une connexion internet stable
  2. Dans ODK Collect, allez dans "Envoyer des formulaires finalisés"
  3. Sélectionnez les formulaires à envoyer
  4. Cliquez sur "Envoyer sélectionnés"
  5. Attendez la confirmation d'envoi
  6. Vérifiez sur votre dashboard que les questionnaires apparaissent
Synchronisez régulièrement (au moins une fois par jour) pour éviter la perte de données.

Questions fréquentes (FAQ)

Ce portail utilise une authentification simplifiée basée sur votre ID enquêteur/superviseur. Pour des raisons de sécurité, contactez votre superviseur si vous souhaitez modifier vos identifiants.

Les statistiques sont mises à jour toutes les 15 minutes. Si vous venez d'envoyer des questionnaires :

  • Attendez quelques minutes et rafraîchissez la page (F5)
  • Vérifiez que vos questionnaires ont bien été envoyés depuis ODK Collect
  • Assurez-vous d'être connecté avec le bon ID enquêteur

Si le problème persiste après 30 minutes, contactez le support technique.

Les manuels sont disponibles à deux endroits :

  1. Dans votre espace (Enquêteur ou Superviseur), après connexion
  2. Dans la section Notes Techniques

Cliquez simplement sur le bouton "Télécharger" à côté du document souhaité. Le fichier sera enregistré dans votre dossier "Téléchargements".

Oui ! Le portail est entièrement responsive et fonctionne sur tous les appareils :

  • Smartphones (Android et iOS)
  • Tablettes
  • Ordinateurs (Windows, Mac, Linux)

Pour une meilleure expérience sur mobile, nous recommandons d'utiliser les navigateurs Chrome, Firefox ou Safari.

Toutes vos données de collecte sont conservées pendant toute la durée de l'enquête, plus une période d'archivage de 5 ans conformément aux réglementations en vigueur.

Vos données de connexion au portail sont stockées localement dans votre navigateur et peuvent être supprimées à tout moment en vous déconnectant.

Si vous rencontrez ce message d'erreur :

  1. Vérifiez votre connexion internet
  2. Actualisez la page (F5 ou Ctrl+R)
  3. Videz le cache de votre navigateur
  4. Essayez avec un autre navigateur
  5. Vérifiez que votre ID est au bon format (ex: ENQ001 ou SUP001)

Si le problème persiste, contactez le support technique avec une capture d'écran de l'erreur.
